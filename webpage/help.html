General:

In games, use Arrow Keys to control direction, and Space to jump/fire.

Games usually require to press Space or 1 in the intro screen to start the actual game.

KC85/2, /3 and /4:

Esc is mapped to the KC85's STOP key, use this to get out of 'modes' (e.g. BASIC's AUTO mode).

Use Shift+Backspace to clear the screen.

ZX Spectrum:

In games that require to select a joystick type, select Kempston.

C64:

The C64 emulation currently only emulates TAP files which are loaded in 'realtime', so some patience is needed :)

After the first short loading phase, when Found [Game Name] is displayed, you can press Space to proceed to the next phase.

Some games require you to type in the RUN command after an initial loading phase (eventually I will automate all this).

In Boulderdash C64, when the first animated into screen appears, press F1 to proceed, then Space to start the game.
